<common_pitfalls>
Based on 03-RESEARCH.md:
1. Time Unit Confusion: OTel uses nanoseconds, UI uses milliseconds. Always convert: durationMs = (end_time - start_time) / 1_000_000
2. Missing Parent Span Edge Case: Child spans may reference parent_id not in captured data. Add cycle protection and depth cap.
3. Attribute Type Variation: OTel attributes can be strings, numbers, booleans, arrays. Normalize all to strings during transformation.
4. Stale Data: Always re-transform from source spans when spans change — don't cache transformed data.
</common_pitfalls>

  <name>Task 1: Create span transformation library</name>
  <files>src/lib/spanTransform.ts</files>
  <action>
    Create spanTransform.ts with functions to transform raw OTel spans into TraceSpan format:
    
    1. **deriveStatus(span):** Map OTel status_code + duration to 'ok' | 'error' | 'warning'
       - ERROR status_code → 'error'
       - duration > 100ms → 'warning' 
       - Otherwise → 'ok'
    
    2. **calculateDepth(span, spanMap, visited):** Calculate nesting depth from parent_id
       - Root spans (no parent_id) → depth 0
       - Use recursion with cycle protection (visited Set)
       - Cap depth at 10 to prevent infinite recursion on malformed data
       - If parent not found in spanMap, treat as root (depth 0)
    
    3. **normalizeAttributes(attributes):** Convert all attribute values to strings
       - Array values: join with ', '
       - Other values: String(value)
       - Handle null/undefined → empty string
    
    4. **transformSpans(rawSpans):** Main transformation function
       - Calculate traceStartNs = min(start_time)
       - Calculate traceEndNs = max(end_time)
       - Build spanMap for parent lookups
       - Map each raw span to TraceSpan with:
         - id: span.context.span_id
         - name: span.name
         - service: span.attributes['service.name'] || 'unknown-service'
         - durationMs: (end_time - start_time) / 1_000_000 (round to 2 decimals)
         - offsetMs: (start_time - traceStartNs) / 1_000_000
         - status: deriveStatus(span)
         - attributes: normalizeAttributes(span.attributes)
         - depth: calculateDepth(span, spanMap)
    
    5. **getTraceId(rawSpans):** Extract trace_id from first span
    
    6. **getTotalDurationMs(rawSpans):** (traceEndNs - traceStartNs) / 1_000_000
    
    Export all functions. Add comprehensive JSDoc comments. Handle edge cases explicitly (empty spans array → throw, invalid timestamps → fallback to 0).
