<objective>
Two performance improvements:

1. **Pyodide loading stages**: The worker posts `{ type: 'loading-stage', stage, total, label }` messages between init steps. `useCodeRunner` surfaces a `loadingLabel` string. `ValidationPanel` shows this label on the Check Code button while `!isWorkerReady`.

2. **Monaco lazy load**: Wrap `CodeEditor` in `React.lazy` + `Suspense` in `App.tsx`. This defers Monaco's large JS bundle until the user navigates to a case — the homepage loads without it.

Purpose: Pyodide init can take 5–15 seconds on cold start; stage labels reduce perceived anxiety. Monaco lazy loading improves first-load time on the homepage.
Output: Modified worker + hook + ValidationPanel for stages; lazy CodeEditor in App.tsx.
</objective>
